
# Configuracao de Credenciais OAuth 2.0

## Passo 1: Acessar o Console
Abra: https://console.cloud.google.com/apis/credentials?project=automation-sheets-drive

## Passo 2: Criar Credenciais OAuth 2.0
1. Clique em "Criar Credenciais" > "ID do cliente OAuth 2.0"
2. Tipo de aplicacao: "Aplicacao Desktop"
3. Nome: "Google Sheets Drive Automation"
4. Clique em "Criar"

## Passo 3: Baixar Credenciais
1. Clique no icone de download (seta para baixo) ao lado das credenciais criadas
2. Salve o arquivo como "credentials.json" na raiz deste projeto
3. Substitua o arquivo credentials.json atual

## Passo 4: Testar
Execute: python cli.py auth login
